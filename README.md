Python Assignment " Causal Inference and Text Analytics "

General aim
This assignment has two connected parts.

1. In Exercise 1, you will estimate causal effects using an autoencoder in the spirit of matrix
completion. You will work in the potential outcomes framework. For treated units, the missing
values after treatment are interpreted as the unobserved no-treatment potential outcomes. You
will estimate these missing outcomes with an autoencoder and then estimate treatment effects.

3. In Exercise 2, you will analyze short text notes describing each unit. You will connect the
text exercise to the causal exercise by using text to predict which units appear to have high
estimated treatment effects.
